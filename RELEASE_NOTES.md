## 🧠 LocalMind v1.0.0 — Initial Release

**Run powerful AI models entirely on your Android device — no cloud, no API keys, no subscriptions.**

### ✨ Highlights

- 🔒 **100% On-Device AI** — All inference runs locally via llama.cpp
- 💬 **Beautiful Chat UI** — Material 3 with Markdown rendering & syntax highlighting
- 📥 **Model Hub** — Browse & download GGUF models from Hugging Face
- ⚡ **Real-Time Streaming** — Token-by-token with optimized TTFT
- 📁 **RAG / Document Chat** — Upload PDFs for context-aware conversations
- 🔧 **Advanced Settings** — Temperature, top-p, top-k, repeat penalty, stop words
- 📂 **Collections** — Organize chats into collections
- 🌐 **Multi-Language** — Full localization support
- 🔐 **Biometric Lock** — Secure with fingerprint/face
- 🎨 **Dark & Light Themes** — Material You dynamic theming

### 📥 Download

| File | Description |
|------|-------------|
| `LocalMind-v1.0.0-universal.apk` | Universal APK (arm64-v8a) — Install directly on any 64-bit Android device |

### 📱 Requirements

- Android 8.0 (API 26) or higher
- 64-bit ARM device (arm64-v8a)
- 4GB+ RAM recommended (6GB+ for 7B models)

### 🛠️ Build from Source

```bash
git clone https://github.com/tk85457/LocalMind.git
cd LocalMind
./gradlew assembleRelease
```
